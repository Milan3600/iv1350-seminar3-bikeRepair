# IV1350 – Object Oriented Design  
## Seminar 3 – Implementation

This repository contains the implementation for Seminar 3 in the IV1350 course. It includes both the program implementation and unit tests.

## Project Members
- Milan Petrovic  
- Ivan Petrovic

The program is structured into five layers:
- **startup** – Contains the `Main` class with the entry point of the program.
- **view** – Contains the `View` class, representing the entire user interface.
- **controller** – Contains the `Controller` class, which handles the program flow.
- **model** – Contains the domain objects and business logic.
- **integration** – Contains the registry creator and external system interfaces (registries and printer).

In total, there are **54 tests** and **9 test classes**.
